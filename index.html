1. What is the typical convergence rate of gradient descent on a smooth and strongly convex function?
  1. Sublinear O(1/t)
  2. Linear O(ρ^t)
  3. Faster Sublinear O(1/t²)
  4. Quadratic O(ρ^2t)
Show Answer & Analysis
Correct Answer: B Analysis: For a function that is both smooth and strongly convex, gradient descent converges linearly. Sublinear O(1/t) applies to functions that are only convex.
2. According to the definition of a Markov State, a state S_t is considered Markov if and only if:
  1. It contains the complete history of all past actions and observations.
  2. The environment is fully deterministic.
  3. The future is independent of the past given the present.
  4. The agent knows the exact absolute location of all objects.
Show Answer & Analysis
Correct Answer: C Analysis: The core of the Markov property is that P[S_{t+1}|S_t] = P[S_{t+1}|S_1,..,S_t]. Once the current state is known, the history may be thrown away because the state is a sufficient statistic of the future.
3. What does the "Reward Hypothesis" state in Reinforcement Learning?
  1. Rewards must always be immediate to be effective.
  2. All goals can be described by the maximization of expected cumulative reward.
  3. Agents must balance positive and negative rewards equally.
  4. A reward is only given when a task is fully completed.
Show Answer & Analysis
Correct Answer: B Analysis: The Reward Hypothesis posits that any goal-directed behavior can be formalized as the maximization of the sum of a scalar feedback signal (expected cumulative reward).
4. In a Partially Observable Markov Decision Process (POMDP), the agent state is:
  1. Exactly equal to the environment state.
  2. Constructed by the agent (e.g., using complete history or RNNs) because the environment is indirectly observed.
  3. Always fixed and cannot be updated.
  4. Irrelevant to calculating the expected reward.
Show Answer & Analysis
Correct Answer: B Analysis: In partial observability (like a poker playing agent only seeing public cards), the agent state ≠ environment state. The agent must construct its own state representation using history, beliefs, or neural networks.
5. Which of the following is NOT one of the major components typically found inside an RL Agent?
  1. Policy
  2. Value Function
  3. Model
  4. Supervised Labels
Show Answer & Analysis
Correct Answer: D Analysis: An RL agent may include a Policy (behavior function), a Value Function (how good a state/action is), and a Model (agent's representation of the environment). Supervised labels belong to Supervised Learning, not RL.
6. The fundamental difference between "Reinforcement Learning" and "Planning" is that in Planning:
  1. The agent must learn directly from interactive game-play.
  2. The rules of the game are completely unknown.
  3. A perfect model of the environment is already known to the agent.
  4. The agent focuses exclusively on exploration rather than exploitation.
Show Answer & Analysis
Correct Answer: C Analysis: In Planning, a model is known, and the agent performs computations (deliberation, tree search) with its model without external interaction. In RL, the environment is initially unknown and requires interaction.
7. In the context of RL trade-offs, trying a new restaurant instead of going to your favorite place is an example of:
  1. Exploitation
  2. Exploration
  3. Temporal Difference Learning
  4. Dynamic Programming
Show Answer & Analysis
Correct Answer: B Analysis: Exploration finds more information about the environment by trying new actions, whereas exploitation exploits known information to maximize immediate reward (going to the known favorite).
8. An algorithm that estimates how "good" a state or action is, and simply picks the action with the highest estimated value (greedy approach), is classified as:
  1. Value Based
  2. Policy Based
  3. Actor Critic
  4. Model Based
Show Answer & Analysis
Correct Answer: A Analysis: Value-based agents (like Q-Learning) rely on the Value Function (V(s) or Q(s,a)) to dictate behavior, unlike policy-based agents that directly learn a mapping from states to actions.
9. What does it mean if an RL agent is "Model-Free"?
  1. It uses deep neural networks instead of traditional machine learning models.
  2. It calculates the explicit transition function P(s'|s,a) before taking an action.
  3. It learns purely through trial and error without trying to learn how the environment works (treats it as a black box).
  4. It cannot solve Markov Decision Processes.
Show Answer & Analysis
Correct Answer: C Analysis: A model-free agent doesn't care how the environment works (doesn't learn P or R functions explicitly); it learns the value of actions directly from experience.
10. The core definition of an "Off-Policy" learning algorithm is that:
  1. It evaluates and improves the exact same policy that is used to make decisions.
  2. The Target Policy being optimized is different from the Behavior Policy generating the data.
  3. It discards data immediately after a single update.
  4. It can only be used with discrete action spaces.
Show Answer & Analysis
Correct Answer: B Analysis: Off-policy algorithms (like Q-learning and DQN) learn from data generated by a different behavior policy or historical data, allowing for high data efficiency via Experience Replay.
11. Which of the following is a known disadvantage of Q-Learning?
  1. It requires a perfect simulator of the environment.
  2. It only works on deterministic environments.
  3. It suffers from an overestimation bias and can be too optimistic during training.
  4. It is an on-policy algorithm, meaning it has poor sample efficiency.
Show Answer & Analysis
Correct Answer: C Analysis: Because Q-learning uses the maximum Q-value of the next state in its update rule, it is prone to overestimation bias, making it overly optimistic.
12. SARSA is distinct from Q-Learning in that SARSA is:
  1. Model-based
  2. Off-policy
  3. On-policy
  4. Policy-based
Show Answer & Analysis
Correct Answer: C Analysis: SARSA is a Value-based, Model-free, On-policy algorithm. It learns from the actual path taken by the behavior policy, making it safer in hazardous environments compared to Q-Learning.
13. How does Deep Q-Network (DQN) break the temporal correlation of training data to improve learning stability?
  1. By using the REINFORCE algorithm.
  2. By using an Experience Replay Buffer.
  3. By performing Monte Carlo Tree Search.
  4. By restricting the state space to low dimensions.
Show Answer & Analysis
Correct Answer: B Analysis: DQN utilizes an Experience Replay Buffer to store (s, a, r, s') tuples and samples them randomly, which breaks the correlation between consecutive experiences and stabilizes training.
14. Proximal Policy Optimization (PPO) is widely used as an industry standard. How is it categorized?
  1. Value-based, Model-based, Off-policy
  2. Policy-based, Model-free, Off-policy
  3. Actor-Critic (Policy-based), Model-free, On-policy
  4. Hybrid, Model-based, Planning
Show Answer & Analysis
Correct Answer: C Analysis: PPO is an Actor-Critic architecture that operates Model-free and On-policy. It constraints the policy update so it does not deviate too far from the one that collected the data.
15. Soft Actor-Critic (SAC) achieves state-of-the-art performance for continuous control tasks largely due to:
  1. Using look-ahead planning like AlphaZero.
  2. Solving a large system of equations dynamically.
  3. Using entropy regularization for superior exploration.
  4. Discarding samples after a single update for stability.
Show Answer & Analysis
Correct Answer: C Analysis: SAC is an Actor-Critic, Model-free, Off-policy algorithm that integrates entropy regularization into the reward to encourage the policy to explore as broadly as possible while succeeding at the task.
